## Instructions

### Development

#### TMDB API Read Access Token

Copy the `env.sample` as `.env` and fill the `TMDB_API_READ_ACCESS_TOKEN` with your token from https://www.themoviedb.org/settings/api

#### Running development server

```sh
bin/dev
```

#### Running code linter

```sh
bin/lint
```

#### Running unit tests

```sh
bin/test_unit
```

#### Running e2e tests

```sh
bin/test_e2e
```
